# AMNXT DIGITAL — Next-Gen IT & Digital Marketing Website

Production-ready, multilingual, SEO-optimized Next.js 16 website.

## Quick Start
```bash
npm install && npm run dev
```
Open http://localhost:3000 → auto-redirects to /en

## Pages
- `/[lang]` — Home (Hero, Stats, Services, Testimonials, FAQ, CTA)
- `/[lang]/about` — Mission, Vision, Values, Timeline
- `/[lang]/services` — 8 services with features
- `/[lang]/projects` — Filterable grid + case study detail pages
- `/[lang]/team` — Team cards + full member profiles
- `/[lang]/contact` — Consultation form + map

## Languages: English (/en) | Hindi (/hi) | Arabic (/ar, RTL)

## Stack: Next.js 16 · TypeScript · Tailwind CSS v4 · lucide-react
